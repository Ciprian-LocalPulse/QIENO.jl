API Reference

Entropic Dynamics

QIENO.EntropicOperator — Type
EntropicOperator(coupling; temperature=1, dissipation=0.05, noise=0.01)

Thermodynamically constrained operator for a graph-coupled stochastic system. coupling is normally a positive-semidefinite graph or hypergraph Laplacian.

source

Return the instantaneous non-negative entropy-production surrogate.

source

Compute deterministic drift under dissipation, coupling, and external forcing.

QIENO.evaluate_stability — Function
evaluate_stability(op, x)

Map entropy production to a bounded score in (0, 1]; larger values indicate lower instantaneous dissipation relative to state energy.

source
